This package defines and implements various transport mechanisms for the EPP Protocol. Currently, it includes the support for TCP transport with TLS.

Summary of API Changes Since EPP-04
EppChannel No change.
EppMessageUtil The com.neulevel.epp.transport.EppMessageUtil.eppMessageSize property and associated access methods have been added for supporting EPP-04 TCP transport mapping:
  • eppMessageSize
  • getEppMessageSizeFetchFlag()
  • getEppMessageSizeWriteFlag()
  • setEppMessageSizeFlags()
The following utility methods have been added:
  • putEppPayload()
EppSession No change.
